A modern, performant portfolio site built with **Next.js 14** and the **App Router**. Features a **custom design system** with mid-century inspired aesthetics and smooth animations.

## Technical Features

**Content Management:**
- **MDX-based blog** with syntax highlighting
- **Markdown experience entries** for easy content updates
- **Gray-matter frontmatter** parsing for metadata

**Design & Animation:**
- **Custom Tailwind configuration** with brand colors
- **Framer Motion animations** with scroll-triggered reveals
- **Responsive design** optimized for all devices
- **Performance-first** approach with Next.js optimization

**Developer Experience:**
- **TypeScript throughout** for type safety
- **Component-driven architecture** for maintainability
- **Git-based content workflow** for version control
